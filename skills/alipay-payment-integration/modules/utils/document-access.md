# 文档访问工具

## 概述

文档访问工具提供了访问支付宝在线文档的标准化方法，包括curl命令的使用、文档的递归访问和内容解析。

## 核心功能

### 基本访问

使用curl命令访问支付宝在线文档：

```bash
# 示例：获取当面付文档
curl -sL "https://ideservice.alipay.com/cms/site/0izcu3"
```

### 递归访问

文档页面内包含的链接需要递归访问以获取完整内容：

1. 首先访问主文档 URL
2. 解析文档中的链接（产品介绍、接入准备、接口文档等）
3. 递归访问这些链接获取详细内容

```bash
# 访问当面付子链接示例
curl -sL "https://ideservice.alipay.com/cms/site/0izal0"   # 产品介绍
curl -sL "https://ideservice.alipay.com/cms/site/0izal1"   # 接入准备
```

### 文档解析

解析文档内容，提取关键信息：

- 接口参数和示例代码
- 接入步骤和注意事项
- 错误码和解决方案

## 最佳实践

- 使用缓存机制减少重复请求
- 设置合理的超时时间，避免网络问题导致的阻塞
- 处理HTTP错误和网络异常
- 记录访问日志，便于问题排查
