# 预授权支付

## 产品介绍

预授权支付是支付宝为需要冻结资金或信用额度的场景提供的支付产品，适用于先冻结资金或信用额度，按实际消费扣款，剩余解冻归还的场景。

## 适用场景

- 酒店民宿、传统租车、分时租赁、单车租赁、充电宝、雨伞、3C数码/手机/相机租赁等

## 核心 API

- `alipay.fund.auth.order.app.freeze`：预授权冻结接口

## 在线文档

- [预授权支付文档](https://ideservice.alipay.com/cms/site/0j0lyx)

## 接入步骤

1. **获取SDK**：根据开发语言选择合适的SDK
2. **配置参数**：设置应用ID、私钥等参数
3. **调用接口**：使用`alipay.fund.auth.order.app.freeze`接口发起预授权冻结
4. **处理结果**：处理冻结结果，确保冻结成功
5. **扣款或解冻**：根据实际消费情况，使用相应接口进行扣款或解冻
6. **异步通知**：接收并处理支付宝的异步通知

## 注意事项

- 必须使用服务端签名，私钥不能存放在客户端
- 必须验证异步通知的签名，确保通知来自支付宝
- 预授权冻结后，需要根据实际消费情况进行扣款或解冻
