# 商家扣款

## 产品介绍

商家扣款是支付宝为周期性自动扣款场景提供的支付产品，适用于用户签约授权后，商家主动发起周期性扣款的场景。

## 适用场景

- 会员包月、自动续费、定期还款等

## 核心 API

- `alipay.trade.app.pay`：支付并签约接口
- `alipay.trade.pay`：后续扣款接口

## 在线文档

- [商家扣款文档](https://ideservice.alipay.com/cms/site/0j0g6k)

## 接入步骤

1. **获取SDK**：根据开发语言选择合适的SDK
2. **配置参数**：设置应用ID、私钥等参数
3. **首次支付**：使用`alipay.trade.app.pay`接口进行支付并签约
4. **后续扣款**：使用`alipay.trade.pay`接口发起后续周期性扣款
5. **处理结果**：处理扣款结果，确保交易成功
6. **异步通知**：接收并处理支付宝的异步通知

## 注意事项

- 必须使用服务端签名，私钥不能存放在客户端
- 必须验证异步通知的签名，确保通知来自支付宝
- 商家扣款产品已于2026年4月2日完成产品升级，本文仅支持最新版本的接入
- 商家扣款当前只支持周期性扣款模式，用户主动免密支付场景暂不支持接入
