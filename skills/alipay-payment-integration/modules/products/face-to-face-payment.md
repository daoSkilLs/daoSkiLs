# 当面付

## 产品介绍

当面付是支付宝为线下实体商户提供的收款产品，适用于用户出示付款码，商家用扫码枪扫码收款的场景。

## 适用场景

- 便利店、商超综合体、餐饮店、医院、学校、电影院、旅游景区等实体门店

## 核心 API

- `alipay.trade.pay`：当面付交易支付接口

## 接入步骤

1. **获取SDK**：根据开发语言选择合适的SDK
2. **配置参数**：设置应用ID、私钥等参数
3. **调用接口**：使用`alipay.trade.pay`接口发起支付
4. **处理结果**：处理支付结果，确保交易成功
5. **异步通知**：接收并处理支付宝的异步通知

## 注意事项

- 必须使用服务端签名，私钥不能存放在客户端
- 必须验证异步通知的签名，确保通知来自支付宝
- 支付结果以异步通知或查询接口为准，不能仅依赖前台结果
