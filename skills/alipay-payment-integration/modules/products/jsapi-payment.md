# JSAPI支付

## 产品介绍

JSAPI支付是支付宝为支付宝小程序提供的支付产品，适用于支付宝小程序内调起支付宝收银台完成支付的场景。

## 适用场景

- 支付宝小程序内购物、服务购买等场景

## 核心 API

- `alipay.trade.create`：创建交易订单接口
- `my.tradePay`：小程序内调起支付接口

## 在线文档

- [JSAPI支付文档](https://ideservice.alipay.com/cms/site/0izg0f)

## 接入步骤

1. **获取SDK**：在支付宝小程序中集成支付能力
2. **配置参数**：设置应用ID、私钥等参数
3. **调用接口**：使用`alipay.trade.create`接口创建交易订单
4. **调起支付**：使用`my.tradePay`接口在小程序内调起支付
5. **处理结果**：处理支付结果，确保交易成功
6. **异步通知**：接收并处理支付宝的异步通知

## 注意事项

- 必须使用服务端签名，私钥不能存放在客户端
- 必须验证异步通知的签名，确保通知来自支付宝
- 支付结果以异步通知或查询接口为准，不能仅依赖客户端返回的结果
