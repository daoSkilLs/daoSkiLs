# App支付

## 产品介绍

App支付是支付宝为原生App提供的支付产品，适用于原生iOS/Android/鸿蒙App内调起支付宝付款的场景。

## 适用场景

- 原生移动应用内支付场景

## 核心 API

- `alipay.trade.app.pay`：App支付接口

## 在线文档

- [App支付文档](https://ideservice.alipay.com/cms/site/0izsn4)

## 接入步骤

1. **获取SDK**：下载并集成支付宝App支付SDK
2. **配置参数**：设置应用ID、私钥等参数
3. **调用接口**：使用`alipay.trade.app.pay`接口生成支付参数字符串
4. **调起支付**：在App中调起支付宝客户端进行支付
5. **处理结果**：处理支付结果，确保交易成功
6. **异步通知**：接收并处理支付宝的异步通知

## 注意事项

- 必须使用服务端签名，私钥不能存放在客户端
- 必须验证异步通知的签名，确保通知来自支付宝
- 支付结果以异步通知或查询接口为准，不能仅依赖客户端返回的结果
- App未安装支付宝客户端时可降级H5支付
