# 手机网站支付

## 产品介绍

手机网站支付是支付宝为移动网页提供的支付产品，适用于手机浏览器H5页面内唤起支付宝App或网页收银台的场景。

## 适用场景

- 移动端网页内支付等场景

## 核心 API

- `alipay.trade.wap.pay`：手机网站支付接口

## 在线文档

- [手机网站支付文档](https://ideservice.alipay.com/cms/site/0izne3)

## 接入步骤

1. **获取SDK**：根据开发语言选择合适的SDK
2. **配置参数**：设置应用ID、私钥等参数
3. **调用接口**：使用`alipay.trade.wap.pay`接口生成支付链接
4. **跳转支付**：引导用户跳转到支付宝支付页面
5. **处理结果**：处理支付结果，确保交易成功
6. **异步通知**：接收并处理支付宝的异步通知

## 注意事项

- 必须使用服务端签名，私钥不能存放在客户端
- 必须验证异步通知的签名，确保通知来自支付宝
- 支付结果以异步通知或查询接口为准，不能仅依赖同步跳转结果
