# 订单码支付

## 产品介绍

订单码支付是支付宝为商户提供的生成二维码，用户扫码付款的产品。适用于商家生成二维码，用户打开支付宝扫码付款的场景。

## 适用场景

- 商品售卖、媒体广告支付等场景

## 核心 API

- `alipay.trade.precreate`：预创建交易订单接口

## 在线文档

- [订单码支付文档](https://ideservice.alipay.com/cms/site/0izg0z)

## 接入步骤

1. **获取SDK**：根据开发语言选择合适的SDK
2. **配置参数**：设置应用ID、私钥等参数
3. **调用接口**：使用`alipay.trade.precreate`接口生成订单二维码
4. **展示二维码**：将生成的二维码展示给用户
5. **查询结果**：使用`alipay.trade.query`接口查询支付结果
6. **异步通知**：接收并处理支付宝的异步通知

## 注意事项

- 必须使用服务端签名，私钥不能存放在客户端
- 必须验证异步通知的签名，确保通知来自支付宝
- 支付结果以异步通知或查询接口为准，不能仅依赖二维码扫描状态
